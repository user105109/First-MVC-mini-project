# Stagiaire Management (MVC PHP)

A simple MVC-based CRUD application built with plain PHP to manage "stagiaires" (interns).  
This project was created to understand the MVC architecture, separation of concerns, and basic DAO/database interaction.

---

## 🚀 Features

- List all stagiaires
- Add a new stagiaire
- Update stagiaire information
- Delete stagiaire
- Simple MVC structure (Controller / Model / View)

---

## 🧱 Tech Stack

- PHP (Vanilla)
- MySQL
- PDO for database connection
- Basic MVC architecture

---

## 📁 Project Structure


controllers/ → Application logic (Controllers)
models/ → Database operations (DAO / Models)
views/ → HTML templates
index.php → Entry point (router)


---

## ⚙️ Setup Instructions

1. Clone the repository:

git clone <repo-url>


2. Import the database:
- Create a database named `ecole`
- Import the SQL file (if provided)

3. Configure database connection:
- Update credentials in the connection file if needed

4. Run the project:
- Use a local server (XAMPP / WAMP / Laragon)
- Open `index.php` in browser
